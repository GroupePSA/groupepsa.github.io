### `Car.GetBatteryLevel()`

| **Description** | Gets the Battery level in percentage.
| **Response** | *Number*  Remaining battery percentage.
| **Parameter**   | *Void*

#### EXAMPLE

```javascript
var BatteryLevel = Car.GetLevel();
// Batteryautonomy = 5 %
```

*Appeared in Software version 40.03.51.50*
