### `Radio.TA()`

| **Description** | Returns true if traffic announcement is available.
| **Response** | *boolean*  Traffic announcement.
| **Parameter**   | *Void*

#### Example

```javascript
// Get the Radio TA
var RadioTA = Radio.TA()
// RadioTA = false
```
