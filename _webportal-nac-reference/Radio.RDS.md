### `Radio.RDS()`

| **Description** | Return true if the current radio stations supports RDS, false otherwise.
| **Response** | *boolean*  RDS support.
| **Parameter**   | *Void*

#### EXAMPLE

```javascript
// Get the Radio RDS 
var RadioRDS = Radio.RDS()
// RadioRDS = true
```
